## 3. Descripción del Proyecto
Sistema diseñado para la administración de préstamos de diversos ítems, incorporando validaciones estrictas, lógica difusa para evaluación de estado y un motor de ventas automáticas con penalización por mora.

## Guia de instalación:

Crear el entorno de desarrollo, IMPORTANTE DONDE ESTE EL ARCHIVO main.py ejecutar este comando:
````
python3 -m venv venv 
````

Instalar libreria para PDF

````
pip install reportlab 
````

Activar entorno
Mac / Linux

````
source venv/bin/activate
````

Windows

````
venv\Scripts\activate
````

Cuando esté activo verás:

````
(venv)
````

Ejecutar app

````
python main.py
````
